# Listas

Las listas guardan varios valores en un solo lugar.

## Que vas a aprender

- Crear una lista
- Acceder a elementos
- Recorrer la lista con un ciclo

## Archivos

- `ejemplo_basico.py`: crear y acceder a una lista.
- `mas_ejemplos.py`: agregar y recorrer una lista.

## Como ejecutar ejemplos

```bash
python app/conceptos/listas/ejemplo_basico.py
python app/conceptos/listas/mas_ejemplos.py
```

## Ejercicios

Lee `EJERCICIOS.md` y ejecuta los archivos de la carpeta `ejercicios/`.
