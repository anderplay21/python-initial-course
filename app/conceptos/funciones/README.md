# Funciones

Las funciones agrupan instrucciones para reutilizarlas.

## Que vas a aprender

- Crear una funcion
- Llamar una funcion
- Retornar un resultado

## Archivos

- `ejemplo_basico.py`: funcion simple que saluda.
- `mas_ejemplos.py`: funcion con numeros y resultado.

## Como ejecutar ejemplos

```bash
python app/conceptos/funciones/ejemplo_basico.py
python app/conceptos/funciones/mas_ejemplos.py
```

## Ejercicios

Lee `EJERCICIOS.md` y ejecuta los archivos de la carpeta `ejercicios/`.
