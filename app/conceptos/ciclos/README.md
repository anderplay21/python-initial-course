# Ciclos

Los ciclos repiten acciones varias veces.

## Que vas a aprender

- Repetir con `for`
- Repetir con `while`

## Archivos

- `ejemplo_basico.py`: ciclo for con range.
- `mas_ejemplos.py`: ciclo while y recorrido de lista.

## Como ejecutar ejemplos

```bash
python app/conceptos/ciclos/ejemplo_basico.py
python app/conceptos/ciclos/mas_ejemplos.py
```

## Ejercicios

Lee `EJERCICIOS.md` y ejecuta los archivos de la carpeta `ejercicios/`.
