# Variables

Una variable guarda un valor para usarlo despues.

## Que vas a aprender

- Guardar datos en variables
- Mostrar variables con `print`
- Cambiar el valor de una variable

## Archivos

- `ejemplo_basico.py`: variables simples y cambio de valor.
- `mas_ejemplos.py`: operaciones y concatenacion con variables.

## Como ejecutar ejemplos

```bash
python app/conceptos/variables/ejemplo_basico.py
python app/conceptos/variables/mas_ejemplos.py
```

## Ejercicios

Lee `EJERCICIOS.md` y ejecuta los archivos de la carpeta `ejercicios/`.
