# Condicionales

Los condicionales permiten tomar decisiones segun una condicion.

## Que vas a aprender

- Usar `if` y `else`
- Agregar mas opciones con `elif`

## Archivos

- `ejemplo_basico.py`: if y else.
- `mas_ejemplos.py`: if, elif, else con varias condiciones.

## Como ejecutar ejemplos

```bash
python app/conceptos/condicionales/ejemplo_basico.py
python app/conceptos/condicionales/mas_ejemplos.py
```

## Ejercicios

Lee `EJERCICIOS.md` y ejecuta los archivos de la carpeta `ejercicios/`.
