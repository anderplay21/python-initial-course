# Documento del curso: Programacion basica con Python

Este repositorio es un material de apoyo para un curso basico de programacion.
Incluye ejemplos, ejercicios y un reto final para practicar paso a paso.

## Razon del repositorio

- Enseñar los conceptos esenciales de programacion con Python.
- Dar ejemplos claros que se puedan ejecutar desde consola.
- Proveer ejercicios guiados para practicar.
- Ofrecer un reto final sencillo para integrar lo aprendido.

## Contenido y organizacion

- `app/main.py`: ejemplo "hola mundo".
- `app/conceptos/`: explicaciones y ejemplos por tema.
- `app/conceptos/*/ejercicios/`: ejercicios practicos.
- `proyecto/`: guia y plantilla para el reto final.

## Actividades del curso

1. Leer la guia general `GUIA_EJEMPLOS_EJERCICIOS.md`.
2. Ejecutar los ejemplos de cada concepto.
3. Resolver los ejercicios de cada concepto.
4. Desarrollar el reto final en la carpeta `proyecto/`.

## Evaluaciones

Las evaluaciones son simples y practicas:

- **Ejercicios por tema**: el estudiante debe completar y ejecutar los ejercicios.
- **Reto final**: el estudiante debe terminar el proyecto y mostrar que funciona.

Se recomienda evaluar con una lista de verificacion:

- ¿El programa se ejecuta sin errores?
- ¿Usa variables, condicionales, ciclos, listas y funciones?
- ¿Tiene entradas con `input()` y salidas claras con `print()`?
- ¿El codigo esta ordenado y es facil de leer?

## Competencias que se desarrollan

- Comprender la estructura basica de un programa en Python.
- Usar variables y tipos de datos simples.
- Tomar decisiones con condicionales.
- Repetir acciones con ciclos.
- Guardar datos en listas.
- Crear y usar funciones.
- Resolver problemas con una secuencia logica.

## Recomendacion para principiantes

- Avanza paso a paso y ejecuta cada archivo.
- No tengas miedo de cambiar valores y probar.
- Lee los mensajes de error con calma y vuelve a intentar.
