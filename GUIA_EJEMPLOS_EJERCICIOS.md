# Guia de ejemplos y ejercicios

Esta guia explica como se organiza el material para aprender Python basico.

## Estructura general

- `app/main.py`: ejemplo de "Hola mundo" por consola.
- `app/conceptos/`: conceptos basicos con ejemplos y ejercicios.

Cada concepto tiene:
- `README.md`: descripcion del concepto.
- `ejemplo_basico.py`: ejemplo sencillo.
- `mas_ejemplos.py`: ejemplos adicionales.
- `EJERCICIOS.md`: ejercicios para practicar.
- `ejercicios/`: carpeta con los archivos de ejercicios.

## Conceptos incluidos

- Variables
- Condicionales
- Ciclos
- Listas
- Funciones

## Como ejecutar ejemplos

Ejemplo para variables:

```bash
python app/conceptos/variables/ejemplo_basico.py
python app/conceptos/variables/mas_ejemplos.py
```

Aplica lo mismo para los demas conceptos, cambiando la carpeta.

Si estas usando Docker, revisa el `README.md` para ver los comandos.

## Como ejecutar ejercicios

Ejemplo para variables:

```bash
python app/conceptos/variables/ejercicios/ejercicio_1.py
python app/conceptos/variables/ejercicios/ejercicio_2.py
```

Aplica lo mismo para los demas conceptos, cambiando la carpeta.

## Consejos para principiantes

- Lee primero el `README.md` del concepto.
- Ejecuta el ejemplo basico y luego los demas ejemplos.
- Intenta resolver los ejercicios cambiando los valores.
